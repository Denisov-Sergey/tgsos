# Похожие решения на GitHub

Подборка готовых проектов для агрегации и сбора постов из Telegram-каналов (для сравнения и идей).

## Агрегаторы и коллекторы

| Проект | Описание | Стек |
|--------|----------|------|
| [kverpoorten/TelegramChannelAggregator](https://github.com/kverpoorten/TelegramChannelAggregator) | Фильтрация сообщений из нескольких чатов и отправка выбранных в один канал | Python, systemd |
| [kevin-glare/telegram_channels_aggregator](https://github.com/kevin-glare/telegram_channels_aggregator) | Агрегация постов из каналов | Python 3, config.json |
| [NikitaPanferov/posts_stealer](https://github.com/NikitaPanferov/posts_stealer) | Сбор постов из нескольких каналов в один канал назначения | Pyrogram, дубликаты, прогресс |
| [qpwedev/telegram-message-collector](https://github.com/qpwedev/telegram-message-collector) | Сбор входящих сообщений в один канал | Pyrogram, config.json (архив 2023) |
| [ahuseyn/tgcollector](https://github.com/ahuseyn/tgcollector) | Веб-GUI для сбора сообщений из каналов | Python, веб-интерфейс |

## Полезные ссылки

- [Telethon](https://github.com/LonamiWebs/Telethon) — библиотека для работы с Telegram API (user client).
- [Pyrogram](https://github.com/pyrogram/pyrogram) — альтернатива Telethon для Telegram MTProto API.
- [my.telegram.org](https://my.telegram.org) — получение API_ID и API_HASH.

Файл обновляется вручную. При добавлении зависимостей или смене стека сверяйтесь с актуальными версиями репозиториев.
